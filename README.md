# 💬 Random Quotes Generator
A simple and interactive Random Quotes Generator built using JavaScript and a public API.

---

## 📌 Features

- Fetches real-time quotes using API
- Generates new quote on button click
- Uses async/await for API handling
- Error handling implemented
- Clean and responsive UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Public Quotes API

---

## 📚 What I Learned

- How to use fetch() to request API data
- Handling JSON responses
- DOM manipulation
- Working with async/await
- Error handling in JavaScript
